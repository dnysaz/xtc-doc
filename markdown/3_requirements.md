# Requirements

## Server (VPS)

- Linux Server, like Ubuntu 20 + , Debian 11 +
- Python 3.8+ and pip
- Port **8080** open (TCP)
- 512 MB RAM minimum

## CLI Client (Mac / Linux)

- macOS 11+ or Linux
- Python 3.8+ and pip
- `make` for install
- UTF-8 terminal
- `xclip` or `xsel` for clipboard (Linux only)

## Windows

- Python 3.10+ from [python.org](https://python.org) — check "Add to PATH"
- Windows Terminal recommended (Microsoft Store)
- Run `.\install.bat` after cloning

> 💡 The cheapest VPS tier at $4–6/month with 512 MB RAM is more than enough for a team of 2–50 people.
